## 一、产品概述

### 1.1 产品定位

**一款专为女性设计的「放大镜面 + 补光拍照」二合一工具 App。** 解决两大真实痛点：
- 外出忘带镜子，手机前置摄像头无法放大检查皮肤妆容细节
- 拍照时需要不同颜色的补光灯来营造氛围感

### 1.2 用户画像

| 维度 | 描述 |
|------|------|
| 核心用户 | 18-40 岁女性 |
| 使用场景 | 外出补妆、检查皮肤、暗光环境自拍、氛围感拍照 |
| 核心诉求 | 看清毛孔/妆容细节、一键补光、操作极简 |
| 使用频率 | 每日多次（化妆镜替代品 + 拍照工具） |

### 1.3 竞品差异

| | 小猫补光灯 | 系统相机 | **镜光 (本产品)** |
|------|:--:|:--:|:--:|
| 补光拍照 | ✅ | ❌ | ✅ |
| 放大镜子模式 | ❌ | ❌ | ✅ **独有** |
| 无按钮全屏镜面 | ❌ | ❌ | ✅ **独有** |
| 冻结帧检查 | ❌ | ❌ | ✅ **独有** |
| 色卡氛围光 | ✅ | ❌ | ✅ |
| 美颜/滤镜 | ❌ | ✅ | ❌（刻意不做） |

#### 3.1.2 功能清单

| 功能 | 描述 | 交互细节 |
|------|------|----------|
| **相机预览** | 前置摄像头实时画面，镜像显示 | 默认前置打开；画面全屏铺满主区域 |
| **切换镜子按钮** | 跳转到界面二（放大镜子模式） | 右上角毛玻璃圆角小按钮，SF Symbol `mirror` 图标 |
| **中心拍照方块** | 预览画面中央的半透明取景参照框 | 方块内构图参考区；方块边缘外由所选补光颜色环绕，模拟环形补光灯照亮人脸 |
| **亮度调节** | 0-100% 屏幕补光强度 | 毛玻璃圆角滑条，iOS 风格 thumb，拖动实时生效 |
| **色卡快捷入口（3 槽位 × 7 色）** | 底部固定 3 个色卡槽，共 7 种预设颜色可左右滑动 | 点击色卡 → 从底部弹出毛玻璃色相/饱和度调节面板；当前选中色卡显示金色边框 + 微发光 |
| **色相调节** | 点击色卡后弹出 360° 色相环 | 拖拽色环取色，实时预览补光颜色变化 |
| **饱和度调节** | 色卡子面板内含饱和度滑块 | 0% = 纯白补光 → 100% = 浓郁彩色氛围光 |
| **全屏补光模式** | 整个屏幕变为当前所选颜色，最大化补光面积 | 点击[⛶]进入全屏，屏幕亮度自动最大；轻触任意处退出；适合极暗环境 |
| **拍照按钮** | 居中大圆形快门按钮 | iOS 毛玻璃环形风格，按下时缩放动画 + 屏幕闪白反馈；中心方块外区域的颜色打在脸上形成氛围补光 |
| **定时拍摄** | 3s / 7s 倒计时 | 长按拍照键弹出定时选项 |
| **镜像修正** | 前置拍出的照片自动水平翻转 | 保存到相册的照片为正常非镜像方向 |

### 3.2 界面二：放大镜子 (Mirror Mode) ⭐ 核心差异化功能

#### 3.2.1 默认状态：全屏无按钮 + 右上角返回键

```
┌──────────────────────────────────────┐
│                            [↩ 返回]  │  ← 右上角毛玻璃返回键（半透明，不抢眼）
│                                      │
│                                      │
│                                      │
│       前置摄像头实时画面               │
│       默认 2x 数码变焦                │
│       镜像显示 (mirrored)             │
│       除返回键外无其他 UI              │
│                                      │
│                                      │
│                                      │
│  ┌────────────────────────────────┐  │  ← 屏幕最底部
│  │ ―  功能框收缩条 (2mm 细线)     │  │     仅半透明细线提示
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘
```

> **核心体验：打开就是一面镜子，只有一个半透明的返回键在右上角，其余区域无遮挡。干净、真实、沉浸。**

#### 3.2.2 展开状态：功能框弹出

**触发方式：** 手指从屏幕底部细线处**向上滑动**（上拉手势）

```
┌──────────────────────────────────────┐
│                            [↩ 返回]  │  ← 返回键始终可见
│                                      │
│       前置摄像头实时画面               │
│       (继续显示，不中断)               │
│                                      │
│  ┌────────────────────────────────┐  │
│  │  补光调节                        │  │
│  │  [暖白] [冷白] [自然光]  ← 3色调  │  │
│  │  亮度: ────●──────── 60%        │  │
│  ├────────────────────────────────┤  │
│  │   [❄️ 冻结帧]     [📷 拍照]     │  │
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘
```

#### 3.2.3 按钮与功能详解

| 按钮 | 位置 | 功能 | 交互细节 |
|------|------|------|----------|
| **↩ 返回** | 右上角常驻 | 返回补光拍照界面 | 毛玻璃半透明圆角按钮；点击后切换至界面一 |
| **📷 拍照** | 功能框内 | 拍摄当前镜子画面 | 保存到相册；拍摄瞬间屏幕闪白 |
| **❄️ 冻结帧** | 功能框内 | 冻结/解冻当前画面 | 画面定格，解放双手检查细节；再次点击或长按画面取消冻结 |
| **补光色温** | 功能框内 | 3 档快速切换 | 暖白(3200K) / 冷白(5600K) / 自然光(4500K) |
| **亮度滑块** | 功能框内 | 屏幕补光强度 0-100% | 拖拽实时生效 |

#### 3.2.4 收缩/展开交互

| 操作 | 结果 |
|------|------|
| **上滑底部细线** | 功能框从底部滑出（动画 0.25s，ease-out） |
| **下滑功能框区域** | 功能框缩回底部，仅保留细线（动画 0.25s，ease-in） |
| **3 秒无操作** | 功能框自动收缩（可配置） |
| **点击画面空白区** | 功能框收缩 |

#### 3.2.5 手势控制（全屏状态下）

| 手势 | 功能 | 说明 |
|------|------|------|
| **双指捏合/扩张** | 调节变焦 1x-5x | 默认 2x；实时显示变焦倍率指示器（2 秒后自动消失） |
| **单指长按（0.5s）** | 冻结/解冻帧 | 长按画面任意处触发；震动反馈 + "画面已冻结"徽章弹出；再次长按取消冻结 |
| **点击右上角返回键** | 返回拍照界面 | 跳转至界面一 |
| **双指双击** | 重置变焦到 2x | 快速回到默认倍率 |

> **⚠️ 手势变更说明：** 冻结帧由"双击画面"改为"长按画面 0.5 秒"。长按更符合直觉（"按住不动→画面定格→仔细看"），且避免与单指点击（收起抽屉）冲突。

#### 3.2.6 变焦指示器

当用户缩放时，画面中央短暂显示（2 秒自动消失）：

```
        ┌──────────┐
        │   2.5x   │
        │  ●━━━━━━━ │  ← 当前倍率 + 进度条
        │  1x    5x │
        └──────────┘
```

## 四、技术规格与卡点应对

### 4.1 变焦策略

| 参数 | 取值 | 依据 |
|------|------|------|
| 默认放大倍率 | 2x | 画质与细节的最佳平衡点 |
| 最大允许倍率 | 5x | 对齐苹果原生相机限制 |
| 推荐倍率提示 | 1x-3x 绿色；3x-5x 橙色 | 超过 3x 提醒画质下降 |
| 上采样阈值 | `videoZoomFactorUpscaleThreshold` | 超过该值后画面开始像素化 |

> **实现细节：** 使用 `AVCaptureDevice.videoZoomFactor`，每次 `lockForConfiguration()` 后设置，并 clamp 到 `min(device.activeFormat.videoMaxZoomFactor, 5.0)`。

### 4.2 屏幕补光

```
方案：UIScreen.main.brightness 控制硬件亮度 + UIView overlay 控制色调
```

| 模式 | 屏幕亮度 | overlay 颜色 | overlay alpha |
|------|----------|-------------|---------------|
| 暖白补光 | 1.0 (最大) | #FFF5E6 | 0.0-1.0 (可调) |
| 冷白补光 | 1.0 (最大) | #F0F8FF | 0.0-1.0 (可调) |
| 自然光 | 1.0 (最大) | #FFFEF5 | 0.0-1.0 (可调) |

> **注意：** 进入镜子模式时保存原始 `UIScreen.brightness`，退出时恢复，避免用户离开 App 后屏幕仍为最大亮度。

### 4.3 机型适配策略

| 机型分组 | 前置像素 | 最大推推荐变焦 | 补光亮度上限 |
|----------|----------|:---:|:---:|
| iPhone 11 / 12 / 13 / 14 / 15 / 16 全系 | 12MP | 3x (推荐) / 5x (硬限制) | 100% |
| iPhone 17 / 17 Pro / 17 Air (2025) | 18MP | 4x (推荐) / 5x (硬限制) | 100% |
| iPhone SE 2/3, iPhone XR/XS | 7MP | 2x (推荐) / 4x (硬限制) | 100% |
| iPad (如有) | 12MP 超广角 | 2x (推荐) / 3x (硬限制) | 100% |

> **实现方式：** 启动时检测 `AVCaptureDevice` 的 `activeFormat` 分辨率，动态设置 maxZoom。

### 4.4 卡点应对清单

| 卡点 | 应对 | 优先级 |
|------|------|:---:|
| 数码变焦画质差 | 默认 2x，超过 3x 显示画质提醒；UI 引导用户靠近而非放大 | P0 |
| 全屏无按钮用户困惑 | 首次进入镜子模式播放 3 秒手势引导动画；仅首次显示 | P0 |
| 老机型画面卡顿 | 使用 `AVCaptureSessionPreset` 降低分辨率以保证 30fps | P1 |
| 屏幕补光耗电 | 退出 App 时恢复亮度；长时间使用(>5min)弹出省电提示 | P2 |
| iPhone 11+ LED 调光需 session | 仅拍照模式使用 LED；镜子模式全用屏幕补光，避坑 | P0 |
| App Store 隐私审核 | `NSCameraUsageDescription` 文案精准描述；隐私政策声明本地不收集数据 | P0 |

## 六、UI/UX 设计规范

### 6.1 整体风格：iOS 原生毛玻璃 (Glassmorphism)

遵循 Apple Human Interface Guidelines，以**半透明 + 背景模糊**为核心设计语言，打造简洁、干净、高级的视觉质感。

| 层级 | 模糊值 | 透明度 | 用途 |
|------|:---:|:---:|------|
| 重度毛玻璃 | blur(35px) saturate(140%) | rgba(10,10,12,0.65) | 底部功能框、调色面板 |
| 中度毛玻璃 | blur(20px) | rgba(255,255,255,0.12) | 按钮、亮度滑条、色卡卡片 |
| 轻度毛玻璃 | blur(10px) | rgba(255,255,255,0.06) | 小徽章、指示器 |
| 无模糊 | — | rgba(0,0,0,0.5) | 全屏遮罩（引导、倒计时） |

### 6.2 配色

| 元素 | 色值 | 用途 |
|------|------|------|
| 主背景 | #000000 | 相机预览区域 / 镜子画面全黑 |
| 功能框背景 | rgba(10,10,12,0.65) + blur(35px) | 毛玻璃控制面板 |
| 收缩指示线 | rgba(255,255,255,0.3) | 底部半透明细线 |
| 按钮主色 | rgba(255,255,255,0.12) + blur(20px) | 毛玻璃按钮底色 |
| 按钮边框 | rgba(255,255,255,0.35) | 毛玻璃按钮边框微光 |
| 激活态 | #FFD700 | 金色 — 色卡选中、按钮高亮 |
| 警告态 | #FB923C (orange-400) | 超过 3x 变焦画质提示 |
| 冻结态 | #38BDF8 (sky-400) | 冻结帧徽章、冻结态按钮 |
| 文字主色 | #FFFFFF | 全界面白色文字 |
| 文字辅色 | rgba(255,255,255,0.5) ~ rgba(255,255,255,0.7) | 辅助说明文字 |

### 6.3 字体 & 图标

- **字体：** SF Pro（系统原生，不引入第三方字体）
  - 标题：SF Pro Text, 17px, 0.25em 字距
  - 正文：SF Pro Text, 12-14px, -0.2px 字距
  - 数字/数据：SF Mono（变焦倍率、百分比）
- **图标：** SF Symbols（Apple 官方图标库，免版权）
  - 镜子模式: `mirror` 或自定义 🪞
  - 返回: `chevron.left`
  - 拍照: `camera` / `camera.fill`
  - 冻结: `snowflake`
  - 亮度: `sun.max` / `sun.min`
  - 全屏: `rectangle.fill`
  - 色相: `circle.lefthalf.filled`
- **最小触控区域：** 44×44pt（遵循 Apple HIG，无障碍标准）

### 6.4 动画

| 动画 | 时长 | 曲线 | 说明 |
|------|:---:|------|------|

### 6.6 设计原则

1. **少即是多** — 仅两个界面，无设置页/登录页/注册页。所有功能在 2 层以内可达
2. **内容优先** — 相机画面铺满屏幕，UI 元素仅在需要时出现，不遮挡主体
3. **毛玻璃一致** — 所有浮层、按钮、面板统一使用玻璃拟态，形成品牌视觉记忆
4. **即时反馈** — 每个操作都有视觉/触觉响应（动画、闪白、震动）
5. **可发现性** — 首次进入提供 3 秒手势引导，关键按钮保留半透明常驻（如镜子返回键）


> ⚠️ 不要声明 `NSPhotoLibraryUsageDescription`（读取权限），除非你确实需要访问相册选图。拍照保存只需 `AddUsage`。

### 7.2 隐私政策

即使不收集数据，也需准备一份隐私政策（可托管在 GitHub Pages），核心声明：

> **镜光 (MirrorGlow) 隐私政策摘要：**
> - 不收集任何个人身份信息
> - 不传输任何相机数据至外部服务器
> - 所有照片保存在用户本地相册，App 不保留副本
> - 无第三方分析/广告 SDK
> - 不使用任何形式的用户追踪

### 7.3 App Store 隐私营养标签

| 数据类型 | 用于追踪 | 关联到身份 | 用途 |

| 照片库 | ❌ | ❌ | App 功能 |

> **全部勾选「不收集」** — 这本身就是一个卖点，可在 App Store 描述中强调"隐私优先"。
